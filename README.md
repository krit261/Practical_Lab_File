Ethical Hacking Practicals – Phishing Awareness Simulation
This repository contains academic practicals focused on cybersecurity awareness and ethical hacking concepts, specifically demonstrating how phishing attacks work in controlled environments.

📚 Practicals Included:
Practical 1: LinkedIn Login Page Simulation (Phishing Awareness)
Practical 2: Snapchat Credential Harvesting Simulation

🎯 Objective:
The main goal of these practicals is to:
Understand how phishing attacks are designed and executed
Learn how attackers mimic real-world login pages
Analyze how user credentials can be targeted
Promote awareness about social engineering attacks

⚠️ Important Disclaimer:
This project is created strictly for educational purposes only as part of cybersecurity learning.
No real data should be collected
These simulations must only be performed in controlled lab environments
Misuse of this knowledge for unauthorized activities is illegal and unethical

🛡️ Learning Outcomes:
Basics of social engineering attacks
Understanding phishing techniques
Awareness of cybersecurity threats
Importance of secure authentication practices

💡 Tools & Environment:
Kali Linux (Virtual Environment)
Educational security tools for demonstration
